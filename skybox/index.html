Quakespasm setup
  1. Put the exported files in your gamedir under gfx/env/.
  2. Use the name without the face suffix or extension, for example sky sample_sky.
  3. Maps can also set worldspawn sky, skyname, or qlsky.
  4. Use the console command sky sample_sky to test or override the current skybox.
  5. PNG, JPEG, and TGA exports use the same rt bk lf ft up dn suffixes.
